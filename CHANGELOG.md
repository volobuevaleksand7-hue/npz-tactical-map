# Changelog

## v1.10.3 - 2026-07-09

**Инфрафикс доставки обновлений (корень «вижу старую версию»).**

- `vercel.json`: `sw.js`, `version.json`, `manifest.webmanifest` получили `max-age=0, must-revalidate`. Раньше `sw.js` подпадал под общее правило `/(.*).(css|js)` с `max-age=3600` → Vercel edge и браузер кешировали сам скрипт service worker на час, браузер не видел новый SW, и старый SW из прошлого релиза продолжал отдавать закешированную оболочку (пользователь застревал на старой версии, напр. v1.9.2 при задеплоенной v1.10.x). Теперь каждый релиз доходит сразу.

## v1.10.2 - 2026-07-09

**Хотфикс: вернул панели радара + переименование кнопки «Радар»→«Радар БПЛА».** Жалоба владельца.

- **Панели радара вернулись** (регрессия v1.10.0): фикс попапов `.map{z-index:auto}` убрал stacking-context карты — при загруженных данных слои Leaflet (маркеры z:600) всплывали в корневой контекст и накрывали panel/feed (z:20). Локально баг не проявлялся: данные радара падали (404), пустой markerPane не перекрывал панель. Откат на `.map{z-index:0}` на обеих картах (`styles.css`, `radar.html`) — слои Leaflet снова заперты внутри карты, панели/карточки сверху; попап у угла панели автопанорамируется в видимую зону.
- **Кнопка «Радар» → «Радар БПЛА»** во всех навигациях: таб-бар главной (`index.html`), навбар радара (`radar.html`), TOP-меню статей (`build-nav.py` → 18 страниц). Таб-бар главной ужат (padding/gap) — на широких экранах метка помещается без наложений.
