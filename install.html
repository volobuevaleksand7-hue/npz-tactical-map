Установите приложение — будьте в курсе
PWA · БЕЗ МАГАЗИНОВ ПРИЛОЖЕНИЙ

«Топливный фронт РФ» ставится прямо из браузера — как обычное приложение, иконкой на экран. Открывается на весь экран, работает и без сети (последняя загруженная карта), обновляется само. Ставить ничего из App Store / Google Play не нужно.

🍏 iPhone / iPad (Safari)

  1. Откройте сайт в Safari (не в Chrome — на iOS ставится только из Safari).
  2. Нажмите кнопку «Поделиться» ⬆️ внизу экрана.
  3. Выберите «На экран «Домой»» (На экран Home).
  4. Нажмите «Добавить» — иконка появится на рабочем столе.

🤖 Android (Chrome)

  1. Откройте сайт в Chrome.
  2. Меню ⋮ (три точки справа вверху).
  3. Выберите «Установить приложение» или «Добавить на главный экран».
  4. Подтвердите — иконка появится в списке приложений.

💻 Компьютер (Chrome / Edge)

  1. Откройте сайт в Chrome или Edge.
  2. Нажмите значок установки ⊕ в правой части адресной строки (или меню → «Установить»).
  3. Приложение откроется в отдельном окне и появится в системе.
🔔 Оповещения. Пока уведомления о тревогах БПЛА приходят в Telegram — бот @BPLAlert_bot (работает и при закрытом приложении). Push-уведомления в самом приложении — скоро.
