❓ Как пользоваться картой

«Топливный фронт РФ» — интерактивная карта нефтеперерабатывающих заводов России, ударов БПЛА по НПЗ, дефицита топлива, сети АЗС и радара угроз. Все данные — оценка по открытым источникам (OSINT), обновляются автоматически несколько раз в сутки и не заменяют официальные оповещения.

🗂 Вкладки и навигация

Вверху сайта — панель вкладок с разделами карты:

  • РОССИЯ — основная карта: НПЗ по всей стране, регионы, удары, слои.
  • КРЫМ — отдельный ТВД: топливная обстановка на полуострове.
  • АЗС — карта заправок, маршрут поездки, фильтр по сетям.
  • РАДАР и НОВОСТИ — открывают отдельные страницы /radar и /news.
  • ИСТОРИЯ — хроника событий с фильтром по типу (удары, ремонты, ограничения, меры).
  • ПРОГНОЗ — сценарии развития кризиса и таблица по горизонтам.
  • ДОХОДЫ — откуда бюджет получает нефтяные доходы и как по ним бьют удары.
  • 📊 АНАЛИТИКА ▾ — выпадающее меню со статьями по регионам («Крым», «Краснодар», «Москва»), объяснялками («Почему нет бензина», «Бензин по талонам») и справочниками (хроника ударов, список НПЗ).

Справа в шапке — время последнего обновления данных, отсчёт до следующего, метка «ОЦЕНКА», кнопка установки приложения 📲, поддержки проекта 🤝 и переключатель светлой/тёмной темы ☀️/🌙.

🗺 Карта: раскраска регионов, маркеры, попапы

Регионы на вкладке «Россия» закрашены по уровню дефицита топлива:

  • Зелёный — норма, топливо доступно свободно.
  • Оранжевый — нагрузка, возможны ограничения и очереди.
  • Красный — критический дефицит.

Переключатель «Сейчас / Прогноз ⚠» над картой меняет раскраску между текущей обстановкой и прогнозной. Ниже — кнопки слоёв: НПЗ, Трубы/порты, 🛣 Дороги, Регионы, 💥 Удары, ⛽ Сеть АЗС, ⚡ Свет, 💰 Цены — каждый можно включить или выключить отдельно.

Точки на карте — заводы (НПЗ) и удары по ним. Клик по метке открывает попап с деталями объекта или события; список НПЗ и легенда цветов дублируются в правой панели.

📱 Мобильная версия

На телефоне боковые панели прячутся, их заменяет плавающая кнопка 📊 (FAB) в углу экрана. Тап по ней открывает выдвижную снизу панель аналитики с вкладками Баланс / Удары / НПЗ / Голоса.

Панель можно свернуть, потянув за ручку-полоску вверху панели вниз, или повторным тапом по кнопке (она превращается в ✕ «Закрыть аналитическую панель»).

Панель вкладок с разделами («Россия», «Крым», «АЗС» и т.д.) на узких экранах закрепляется внизу экрана как нижняя навигация — вместо верхней шапки.

💥 Таймлайн ударов

Под картой на вкладке «Россия» — панель «💥 Удары» с быстрыми периодами: Сегодня, 7 дней, 30 дней, Все. Раздел «Даты» открывает свой диапазон (в т.ч. пресет «Пр. месяц») с полями «от» и «до».

Раздел «Таймлайн» даёт пошаговую перемотку по дням: кнопки ‹ и › — шаг назад/вперёд, ползунок — быстрый переход к дате, кнопка ▶ — автопроигрывание ударов по дням. Слой «💥 Удары» на карте должен быть включён, иначе точки не отображаются.

⛽ Слои и фильтры АЗС

На вкладке «АЗС» — карта заправок (расположение реальное, по OpenStreetMap; наличие топлива — оценка по сети и региону, не по конкретной колонке).

Блок «🚗 Карта поездки»: быстрые пресеты маршрутов, поля «Откуда» / «Куда» и кнопка «Маршрут» строят линию поездки с заправками по пути; ✕ сбрасывает маршрут.

Блок «Фильтр сети» — пилюли брендов АЗС, по клику включают/выключают показ конкретной сети на карте. Справа — панель с комментариями с мест.

📡 Радар (/radar)

Отдельная страница /radar — живой радар угроз БПЛА и ракетной опасности по регионам, обновляется примерно каждые 5 минут.

Раздел «Типы угроз» (сворачиваемый) поясняет цвета меток: БПЛА, КАБ, FPV, Ракета, Авиация, Взрыв. Рядом — лента «📨 ЛЕНТА» с текстовыми сообщениями источников и ссылка на Telegram-бота тревог.

Блок «ℹ️ О карте тревог» объясняет, что это оценка по OSINT, а не официальное оповещение МЧС.

📲 Установка приложения

Карту можно поставить на телефон или компьютер прямо из браузера — без App Store и Google Play. Кнопка «Установить приложение» в шапке сайта запускает установку (на iPhone работает только через Safari — «Поделиться» → «На экран «Домой»»).

Подробная пошаговая инструкция с разбивкой по iOS / Android / Desktop — на странице /install.

⚠️ Все данные на карте — оценка по открытым источникам (OSINT) и носят справочный характер. Не являются официальным оповещением МЧС или региональных властей.
