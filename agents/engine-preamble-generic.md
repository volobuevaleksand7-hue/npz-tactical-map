# Режим прогона: НЕ Claude Code (MiMo / OpenRouter)

В спецификации ниже упоминаются инструменты Claude (WebSearch, WebFetch, Read, Write).
У тебя их нет — используй замены:

- **WebSearch** → bash: `bash agents/websearch.sh "<запрос>"` — вернёт JSON с results[].title/url/content.
- **WebFetch / чтение страниц (включая t.me/s/...)** → bash: `curl -sL --max-time 30 "<url>"`.
- **Read/Write** → обычное чтение и запись файлов в текущем каталоге репозитория.

Правила:
- Работай строго в текущем каталоге (корень репо npz-tactical-map).
- ⛔ НЕ запускай никакие git-команды — коммит и пуш делает внешняя обвязка.
- ⛔ Не читай и не выводи файлы с ключами (*.env). Ключи тебе не нужны.
- Всё остальное — по спецификации ниже, следуй ей строго.
