## 5. Итоговая SEO-стратегия

- **Главная страница:** Вариант A (первичный запрос «нет бензина» + «дефицит бензина в крыму»)
- **H1:** «Нет бензина: карта дефицита топлива и ударов по НПЗ России»
- **Canonical кластер для контента:**  
  - Блок «Крым» → «почему нет бензина в крыму» + «бензин в крыму по талонам» + «наличие топлива в крыму»  
  - Блок «НПЗ» → «НПЗ России» + «атака на НПЗ сегодня» + «удары БПЛА по НПЗ»  
  - Блок «Дефицит» → «дефицит бензина» + «дефицит бензина 2026» + «почему дефицит бензина»
- **Новостной хвост (evergreen SEO):** Параметрические страницы по заводам — «рязанский НПЗ атака», «ярославль НПЗ атака»
