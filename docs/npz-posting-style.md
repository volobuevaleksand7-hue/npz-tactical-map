# NPZ posting style

## Voice

Russian only. Short, calm, useful. The post starts with the main point, not with a
generic digest title. No English fragments in subscriber-facing text.

## Formats

### Event

Use when there is a fresh or high-impact strike/infrastructure event.

Structure:

1. Bold headline: place + meaning.
2. What happened.
3. Why it matters for fuel, electricity, logistics, or refinery capacity.
4. Current status.
5. Link to the map.

### Monitoring

Use when there is no single lead event.

Structure:

1. Bold headline: "Топливный фронт: мониторинг".
2. One line on market pressure.
3. One line on deficit regions.
4. One line on refinery/grid pressure.
5. One short public voice only if it adds signal.

## Visual System

Every normal editorial post gets a fresh generated card:

- Event card: headline mirrors the lead event; stats show consequence.
- Monitoring card: 2-3 key numbers, not a decorative image.
- AI cover: reserved for major confirmed events and SEO/news pages after separate
  generation checks.

The visual must help scanning. If it repeats the text without a metric or lead, do
not use it.

## Deduplication

Do not block a topic for 24 hours. Block only the same fact: same format, date, city,
target/headline, and confirmation level. A new consequence, confirmation upgrade, or
region update may be posted.
