# Бэклог: 10 идей для развития «Топливного фронта РФ»

> Отложено из аудита от 2026-07-09. Это НЕ фиксы аудита (те — в `docs/audit-fixes.workflow.mjs`),
> а идеи следующего уровня: как сделать сайт современнее и сильнее. Каждая — с оценкой
> усилия и эффекта, чтобы брать по одной, когда дойдут руки.

Легенда: Эффект / Усилие — 🟢 малое · 🟡 среднее · 🔴 большое.

| # | Идея | Эффект | Усилие | Почему стоит |
|---|------|:---:|:---:|------|
| 1 | **Единый генератор шапки+футера** для всех шаблонов (index, radar, news, лендинги) через расширенный `build-nav.py` | 🔴 | 🟡 | Убивает треть находок аудита разом и навсегда: новые страницы наследуют навигацию автоматически. Фундамент для всего остального. |
| 2 | **«АЗС рядом со мной»** — геолокация + фильтр «есть топливо» на карте АЗС | 🔴 | 🟡 | Главный мобильный интент вашего трафика. Конкурент gdebenzin статусы фейкает (Math.sin от OSM id) — честная фича = моментальное преимущество. |
| 3 | **Кастомная 404 как точка входа** — мини-статус + поиск + топ-5 разделов | 🟡 | 🟢 | Новостной трафик приходит по битым ссылкам из соцсетей; сейчас — голая заглушка Vercel, теряем людей. (Базовый вариант уже в workflow-фиксах.) |
| 4 | **Внутренний поиск** (город / НПЗ / дата) — статический JS-индекс по seo-topics + strikes.json, без бэкенда | 🟡 | 🟡 | 80+ страниц — меню перестаёт масштабироваться. Поиск = быстрый путь к нужному городу/заводу. |
| 5 | **Месячные хабы архива** `/news/2026-06/` — сводки месяца + статистика ударов | 🟡 | 🟡 | SEO-страницы под «удары по нпз июнь 2026», разгрузка бесконечной ленты /news, чистая структура. |
| 6 | **Компактный KPI-бар на мобильной главной** (35% · 9 стоит · 63 удара/7д), свайп вверх → полная шторка | 🟡 | 🟢 | Сейчас мобильный пользователь вообще не видит ключевые цифры сайта с первого экрана. |
| 7 | **RSS + новостной sitemap** для сводок | 🟡 | 🟢 | Дёшево; даёт агрегаторы, Дзен-подобные источники, быструю индексацию свежести — ваш главный SEO-актив. |
| 8 | **Виджет-embed радара** (`<iframe>` «угрозы сейчас: N регионов») для форумов и Telegram | 🟡 | 🟡 | Бесплатные обратные ссылки с тематических площадок, сетевой эффект, рост охвата. |
| 9 | **Страница «Методология»** — откуда данные, как считается «стоит/частично», сетка источников | 🟡 | 🟢 | E-E-A-T для YMYL-тематики (прямой сигнал качества Google) + щит от обвинений в неточности. |
| 10 | **Перф-паузы**: self-host шрифтов и leaflet+markercluster, `loading=lazy` на карточки /analytics, ужать og-image (сейчас 400 КБ) | 🟡 | 🟢 | 4 внешних хоста в критическом пути (unpkg + Google Fonts). На мобильном 4G ≈ −1 с к первому рендеру карты. |

## Рекомендованный порядок

1. **Сначала #1** (единая шапка) — это платформа, на ней дешевеет всё остальное.
2. Затем быстрые победы 🟢: #3, #6, #7, #9, #10 — малое усилие, ощутимый эффект.
3. Флагманская фича #2 (АЗС рядом) — когда будет время на 🟡, она дифференцирует продукт.
4. #4, #5, #8 — по мере роста числа страниц и внешнего охвата.

## Не делать пока

- Не строить бэкенд ради поиска/геолокации — всё перечисленное живёт на статике + чистом JS.
- Не переписывать дизайн целиком: три темы (карта/радар/статьи) свести к одной палитре можно
  эволюционно через общий шаблон (#1), а не переделкой с нуля.
