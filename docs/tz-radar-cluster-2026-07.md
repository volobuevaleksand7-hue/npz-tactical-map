- **H1:** `Карта БПЛА и воздушных тревог: угрозы по регионам в реальном времени`
- **Primary:** `карта бпла` (126 535). **Вторичные:** карта бпла онлайн, карта тревог, радар карта бпла, карта воздушных тревог, карта атак бпла, карта бпла в реальном времени.
- **H2-план (5):** 1) Как читать карту-радар (легенда: БПЛА/ракета, регионы, обновление) + CTA → /radar → 2) Что показывает радар (угрозы по регионам НПЗ, свежесть данных) → 3) Воздушная тревога и ракетная опасность — в чём разница (объяснялка, ключ 576+) → 4) Отбой тревоги: как понять (ключ 7 145) → 5) FAQ.
- **FAQ (6):** «Где смотреть карту БПЛА онлайн?» · «Что такое ракетная опасность и воздушная тревога, чем отличаются?» · «Как узнать про отбой тревоги?» · «Откуда данные на карте?» · «Обновляется ли карта в реальном времени?» · «Какие регионы под угрозой БПЛА?»
- **Тон:** строго нейтральный OSINT-мониторинг. Дисклеймер: оценка по открытым источникам, НЕ официально, НЕ инструкция к действию, не замена сигналам МЧС/оповещения.
- **CTA:** крупный «📡 Открыть карту-радар угроз →» на `/radar` (hero + повтор в теле).
- **Реопт `/radar`:** title/description под «карта бпла онлайн», + текстовый блок под H1 (сейчас радар — голое приложение без SEO-текста).
- Реестр: строка в `seo-topics.jsonl` type `tool`, группа «Карты» → авто в меню/хаб/чип. Sitemap — авто (`generate-sitemap.py`).

## Данные для страницы (честно)
Радар показывает ОЦЕНКУ угроз по регионам из открытых источников (`data/radar-state.json`), не официальные оповещения. На карте — регионы НПЗ; гражданский охват шире по тексту, но подчёркиваем: это OSINT-агрегация, не система оповещения.
