# ТЗ: статьи-воронки на карты (АЗС + Радар), июль 2026

**Идея.** Две недоиспользованные карты сайта: (1) **карта АЗС** — вкладка ⛽ в `/` (`/?view=azs`), наличие по сетям/регионам + планировщик поездки «Откуда→Куда»; (2) **радар** `/radar` — угрозы БПЛА/ракет по регионам НПЗ, live. Люди о них не знают. Катаем SEO-страницы, которые бьют в дефицит/«где топливо» → ведут на карту АЗС, и в «атака сегодня/опасность» → ведут на радар. **На каждой странице — крупный, заметный переход на карту (hero-CTA).**

**Энейблер (уже есть, код не трогать):** `app.js:1560` читает `?view=azs` / `#azs` и открывает вкладку карты при загрузке. Значит воронка ведёт на `/?view=azs` без правки защищённого ядра. Deep-zoom по региону/фильтр сети через URL пока НЕ поддержан — v1 ведёт на карту АЗС, юзер фильтрует сам (честно).
